# LOVEGRAM - Instagram Login :bomb:

Installation requirement
--

```
1. pkg install git
2. pkg install php
3. git clone https://github.com/dandyraka/lovegram.git
```

How to run
--
```
1. cd lovegram
2. php login.php
```

Input your instagram Username & Password.

Then copy the TOKEN to Lovegram Login page
